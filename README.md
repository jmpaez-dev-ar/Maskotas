## 🤖 Proyecto Demostrativo: GitHub Copilot + Claude Sonnet 4

**Maskotas** es un sitio web moderno y responsivo dedicado al cuidado y bienestar de las mascotas, desarrollado **exclusivamente como proyecto demostrativo** para ilustrar las capacidades extraordinarias de **GitHub Copilot** trabajando en sinergia con el modelo de IA **Claude Sonnet 4**.

### 🎯 Objetivo Demostrativo
Este proyecto es una **demostración práctica** de cómo la combinación de GitHub Copilot y Claude Sonnet 4 puede revolucionar el desarrollo web moderno:

- ✅ **Acelerar el desarrollo 3-4x**: Sugerencias inteligentes en tiempo real
- ✅ **Implementar funcionalidades complejas**: Galería interactiva con filtros, lightbox y lazy loading
- ✅ **Crear sistemas avanzados**: Formularios con validación en tiempo real y autoguardado
- ✅ **Optimizar el rendimiento**: Debounce, IntersectionObserver y mejores prácticas automáticas
- ✅ **Garantizar accesibilidad**: Implementación automática de estándares WCAG AA
- ✅ **Generar documentación técnica**: 6+ archivos de documentación profesional
- ✅ **Resolver problemas sistemáticamente**: Detección y corrección automática de bugs
- ✅ **Seguir mejores prácticas**: Código limpio, semántico y optimizado desde el primer intento

### 🚀 Desarrollo 100% Asistido por IA
Desarrollado completamente utilizando **HTML5, CSS3 y JavaScript vanilla** con la asistencia integral de:
- **GitHub Copilot**: Sugerencias de código contextuales, autocompletado inteligente y generación de funciones
- **Claude Sonnet 4**: Análisis profundo de código, arquitectura de proyecto, resolución de problemas complejos y documentación técnica

> **NOTA**: Este proyecto demuestra que es posible crear aplicaciones web completas y profesionales utilizando únicamente herramientas de IA, sin sacrificar calidad, rendimiento o mejores prácticas.

## 📦 Cómo Usar

### 🚀 Instalación Rápida

1. **Clonar el repositorio**
   ```bash
   git clone https://github.com/tu-usuario/maskotas.git
   cd maskotas
   ```

2. **Abrir el proyecto**
   - Abre `index.html` directamente en tu navegador, o
   - Usar un servidor local (recomendado):
   ```bash
   # Con Python
   python -m http.server 8000
   
   # Con Node.js (si tienes http-server instalado)
   npx http-server
   ```

3. **Explorar las funcionalidades**
   - Navega por las diferentes secciones
   - Prueba el formulario de contacto con todas sus características
   - Utiliza los atajos de teclado para una experiencia optimizada

### 🧪 Testing y Validación

Para probar todas las funcionalidades del sitio:

1. **Testing Manual**: Abre `docs/testing-manual.html` para una guía completa de testing
2. **Validación HTML**: Verifica en [W3C Validator](https://validator.w3.org/)
3. **Accesibilidad**: Prueba con lectores de pantalla y navegación por teclado
4. **Responsive**: Utiliza DevTools para probar en diferentes dispositivos

## 🔧 Personalización

### Agregar Nuevas Secciones
1. Añade la sección en `index.html`
2. Agrega los estilos correspondientes en `styles.css`
3. Actualiza la navegación si es necesario

### Modificar Colores
Edita las variables CSS en `:root` en el archivo `styles.css`:

```css
:root {
    --color-primary: #tu-color-primario;
    --color-secondary: #tu-color-secundario;
    /* etc... */
}
```

### Agregar Imágenes
1. Coloca las imágenes en la carpeta `img/`
2. Referéncialas en el HTML: `<img src="img/tu-imagen.jpg" alt="Descripción">`
